Thinking Like a Programmer

1. Procedural Thinking

Procedural thinking involves creating perfectly clear and unambiguous instructions for a computer to follow.

2. Abstract Thinking

Abstract thinking means finding similarity, or as programmers would say, generality amongst seemingly different things.

3. Systems Thinking

Systems thinking happens when you break a big problem down into smaller pieces. Programmers do this when they create a plan (often on paper) for how a program will work. It involves big-picture thinking and decision-making about a problem and how different pieces of a program can work together to solve it.

4. Technological Empathy

"Technological empathy" comes in many forms.

5. Debugging

Debugging is a systematic process of relentlessly identifying the cause of a computer program that doesn't work.


Basic HTML Elements

HTML elements are what we use to tell a web browser how to display content on a web page.

Most HTML elements consist of an opening and a closing tag with some content between them.

b tag Makes the 'text' content bold

em tag Makes the 'text' content italic.

HTML Attributes

Some elements accept additional values called attributes in their opening tags. Attributes provide additional information to the browser. Udacity Creates a hyperlink to the href attribute value url and names it Udacity.You also have the div, span and br tags
